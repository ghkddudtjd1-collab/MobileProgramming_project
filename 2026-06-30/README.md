# 2026-07-01 학습 정리

`Class_and_function.ipynb`를 통해 코틀린의 클래스 구조와 람다 함수를 중심으로 학습했다.

## 학습 주제

### 1. 상속과 다형성
- `Shape`, `Circle`, `Rect`, `Ellipse` 예제로 클래스 상속 구조를 익혔다.
- `override`를 사용해 부모 메서드를 재정의하는 방법을 연습했다.
- `List<Shape>`에 서로 다른 하위 클래스를 담아 다형적으로 동작시키는 방식을 확인했다.

### 2. 타입 검사와 형변환
- `is`를 사용한 타입 검사 방법을 학습했다.
- `as`를 이용한 다운캐스팅으로 하위 클래스 고유 메서드(`info`)를 호출했다.
- `when`과 타입 조건을 함께 사용해 객체 종류별로 분기하는 방법을 정리했다.

### 3. 추상 클래스
- 추상 클래스는 직접 객체를 생성할 수 없다는 점을 확인했다.
- 추상 메서드가 포함된 클래스는 `abstract class`로 선언해야 함을 학습했다.
- `Animal2`, `Cat` 예제로 추상 메서드 구현과 일반 메서드 재정의를 연습했다.

### 4. 오버라이딩과 오버로딩
- `override`로 부모의 동작을 재정의하는 오버라이딩을 복습했다.
- 같은 이름의 메서드를 매개변수만 다르게 만드는 오버로딩도 함께 확인했다.

### 5. 인터페이스
- `Flyable` 인터페이스를 통해 추상 메서드와 기본 메서드의 차이를 학습했다.
- `Duck` 클래스에서 인터페이스 구현과 기본 메서드 재정의를 실습했다.

### 6. 클래스 위임
- `Printer` 인터페이스와 `Logger2` 예제로 클래스 위임(`by`) 문법을 익혔다.
- 구현을 직접 다시 작성하지 않고 다른 객체에 기능을 맡기는 방식을 이해했다.

### 7. 람다 함수와 고차 함수
- `val double: (Int) -> Int = { it * 2 }` 형태로 람다를 변수에 저장하는 방법을 학습했다.
- `multiplier` 예제를 통해 함수를 반환하는 함수와 클로저 개념을 연습했다.
- 함수 타입 `(Int) -> Boolean`, `() -> String`을 사용하는 고차 함수 `testFun`으로 람다 전달과 반환을 함께 확인했다.

## 핵심 정리
- 코틀린은 클래스, 추상 클래스, 인터페이스를 통해 다양한 객체지향 구조를 표현할 수 있다.
- `is`, `as`, `when`을 활용하면 타입 기반 제어를 명확하게 작성할 수 있다.
- 람다 함수와 고차 함수를 사용하면 함수를 값처럼 다루는 함수형 프로그래밍 스타일을 적용할 수 있다.
