# 2026-06-26 학습 정리

- Kotlin의 기본 자료형(`Double`, `Float`, `Long`, `Short`, `Char`) 선언 방식과 리터럴 표기법을 정리했다.
- `var`와 `val`의 차이, 타입 변환 규칙, nullable/non-nullable 개념을 예제로 확인했다.
- `lateinit` 사용 조건과 함수 선언 방식, 반환형 `Int`/`Any`, 기본 매개변수 문법을 학습했다.
- `if`를 식처럼 사용하는 방법과 `when`으로 여러 조건을 분기하는 Kotlin 문법을 연습했다.
- `for` 반복문과 급수 계산 예제를 통해 반복 처리와 수식 구현 방식까지 함께 연습했다.
